## âš ï¸ IMPORTANT NOTES FOR USERS

### v1.3.0 Breaking Changes (Server Side)

Users upgrading VideoAnnotator to v1.3.0 should know:

1. **Auth Required by Default**
   - `AUTH_REQUIRED=true` is now default
   - Need to generate token: `videoannotator generate-token`
   - Our client already handles this!

2. **CORS Restrictions**
   - Default: `ALLOWED_ORIGINS=http://localhost:19011`
   - Users need to add their client URL
   - We use `18011` by default - document this!

3. **Persistent Storage**
   - Default: `./storage` (not `/tmp`)
   - Data survives restarts
   - Configurable retention policies

### Document These in Updated Guides

- How to configure CORS for our default port
- How to generate API tokens
- Environment variable reference
